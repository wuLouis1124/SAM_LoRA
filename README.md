### How to run it on your own machine

1. git clone
    ```
   $ git clone https://github.com/xzyun2011/finetune_segment_anything_tutorial.git
   ```

2. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

3. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
